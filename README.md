# Level 1 — eCitizen Login Validation System

## Description
This project is a login validation system that checks username and password. 
It is implemented in Python, Java, and C++.

## Concepts Demonstrated
- Syntax vs Semantics
- Compilation vs Interpretation
- Runtime errors
- Logical errors

## Usage
1. Enter username and password when prompted.
2. Program prints "Access Granted" if credentials match.
3. Program prints "Invalid Credentials" if credentials do not match.

- `level1_ecitizen.py` (Python)
- `level1_ecitizen.java` (Java)
- `level1_ecitizen.cpp` (C++)
